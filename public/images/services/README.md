# Service Images Directory

This directory contains images for the cremation services offered through the RainbowPaws application.

## Naming Convention

- Service images should follow the pattern: `service_[ID].jpg` or `service_[ID].png`
- For example: `service_001.jpg`, `service_002.png`, etc.

## Default Images

- `default.jpg` - Default placeholder for services without an image
- `[category]_default.jpg` - Category-specific default images (e.g., `individual_default.jpg`, `communal_default.jpg`)

## Categories

- Individual Cremation
- Premium Cremation
- Communal Cremation
- Memorial Services
- Additional Services

## Image Requirements

- Recommended size: 800x600 pixels
- Maximum file size: 2MB
- Supported formats: JPG, PNG
